# 凌的博客
## 欢迎
欢迎来咱的博客玩！！！

使用此博客需要一定shell基础。

示例可以见`example/hello-world.js`。在终端中使用`apt install /example/hello-world.js`安装示例程序。

我们有了第一个官方应用程序——伪Node.js！使用`apt install https://furryr.github.io/term-plugins/node.js`安装。

## 重置文件系统

在访问后加上`?reset`即可重置文件系统。

一旦重置文件系统，所做的任何更改都将丢失，请注意。

版本更新时可以使用`apt`安装新应用，也可以直接重置文件系统。

玩坏系统时请务必重置。
